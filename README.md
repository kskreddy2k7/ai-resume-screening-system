# AI Resume Screening System

## Overview
An AI-based resume screening system that uses NLP and Machine Learning
to analyze resumes and rank them based on job requirements.

## Features
- Resume parsing using NLP
- Keyword extraction
- Resume ranking based on relevance
- Automated screening

## Technologies Used
- Python
- NLP
- Machine Learning
- Flask
- Pandas, NumPy

## How It Works
1. Upload resume
2. Extract text using NLP
3. Match skills with job description
4. Rank resumes
